# Production Account

The Production account hosts live workloads and projects available to the public. It allows me to keep finished work and data separate from incomplete projects on the Staging or Development environments, while ensuring a polished state due to testing on staging.

## Purpose
- Runs public‑facing workloads and projects.
- Enforces strict governance and cost‑control boundaries.

## Security
- Access is solely given to myself through AWS IAM Identity Center.
- Root account protected with MFA and unused for daily operations.
